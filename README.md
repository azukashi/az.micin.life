## ➡️ Development

### 🛠️ Requirements

A [node.js](https://nodejs.org) >= 18.x setup with [bun](https://bun.sh) is recommended.

### 📦 Install dependencies

```sh
$ bun install
```

### 🏃 Run it on development server

```sh
$ bun dev
# $ next dev --turbopack
#    ▲ Next.js 15.x.x (Turbopack)
#    - Local:        http://localhost:3000
#    - Network:      http://x.x.x.x:3000
#
#  ✓ Starting...
#  ✓ Ready in 1ms
```
